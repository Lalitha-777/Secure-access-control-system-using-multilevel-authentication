# Secure-access-control-system-using-multilevel-authentication.
Secure Access Control System Using Multilevel Authentication is an embedded system project that enhances security by verifying users through multiple authentication methods before granting access. Developed using Embedded C in Keil, it prevents unauthorized entry, improves reliability.

key features:
1) Fingerprint sensor (R305) for biometric authentication
2) Motor driver to control the locking mechanism
3) LCD display for real-time user feedback and menu navigation
